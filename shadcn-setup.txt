# shadcn/ui Setup Guide

1. **Install dependencies**
    ```
    npx shadcn@latest init (terminal)
    ```

2. **Add shadcn/ui components
   
    npx shadcn@latest add button (terminal) - For Button
    npx shadcn@latest add input  (terminal) - For Input
    npx shadcn@latest add label  (terminal) - For Label
    npx shadcn@latest add form   (terminal) - For Form
   
   
3. Example Adding Form - follow through https://ui.shadcn.com/docs/components/form


# Done!
